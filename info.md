# ThingsBoard Integration

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/custom-components/hacs)

Eine vollständige Home Assistant Custom Component Integration für ThingsBoard IoT-Plattform.

## Features

✅ **Automatische Entity-Discovery** - Erkennt alle Attribute automatisch
✅ **Stündliche Aktualisierung** - Daten werden jede Stunde aktualisiert
✅ **HTTP API** - Nutzt die ThingsBoard HTTP Device API
✅ **Gerätesteuerung** - Setzen von SharedAttributes
✅ **Number-Entities** - Numerische Werte direkt in UI steuerbar
✅ **Services** - Erweiterte Steuerung über Automatisierungen

## Schnellstart

1. Installieren Sie die Integration über HACS
2. Starten Sie Home Assistant neu
3. Gehen Sie zu **Einstellungen** → **Geräte & Dienste**
4. Klicken Sie auf **+ Integration hinzufügen**
5. Suchen Sie nach **ThingsBoard**
6. Geben Sie Host-URL und Device Access Token ein
